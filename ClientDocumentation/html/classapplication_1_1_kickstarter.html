Detailed Description

This class is meant as a bridge between the GUI Controller and the Traffic Light.

The reason for this being the Main class (TLClient) is mainly used to initialize the GUI and its JavaFX controller, thus in order to separate the GUI controller and what actually starts the client backend, this acts like the main method to kickstart the client instead of handling client objects inside the GUI controller.
