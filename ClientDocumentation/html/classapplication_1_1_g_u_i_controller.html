Detailed Description

JavaFX controller to control the Scene Builder created user interface.

Member Function Documentation

void application.GUIController.connect ( ActionEvent  event)

Button event when the button_connect is pressed.

Connects the client by creating the bridge between the Traffic Light and the GUI and sending user interface related information from the host field, the port field and type fields. While also sending the imageview further down the classes.

void application.GUIController.disconnect ( ActionEvent  event)

Button event when the button_disconnect is pressed.

Disconnects the client by stopping the threads and nulling the bridge killing off the client's connection completely. Also reenables the connect button by using the workaround.
